📅 Meeting Tracker Setup

Paste your Google Apps Script web app URL below to connect your sheet. Follow these steps to create it — takes about 2 minutes.

1
Open your Google Sheet → Extensions → Apps Script
2
Delete any existing code and paste in the script provided separately, then click Save (💾)
3
Click Deploy → New deployment → type: Web app → set Who has access to Anyone → click Deploy
4
Copy the Web app URL and paste it below
